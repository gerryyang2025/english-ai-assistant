# Lottie 资源

## `mascot-character.json`

首页墨小灵使用的**纯矢量**角色动画（无位图叠加），来自 [LottieFiles](https://lottiefiles.com/) 公开资源包 `lf20_M9p23l.json`（CDN 抓取后入库，便于离线部署）。格式说明见 [Lottie Animation Community](https://lottie.github.io/) 与规范文档。

替换动画时：保持文件名 `mascot-character.json`，或修改 `js/app.js` 中 `initMoxiaolingMainLottie` 的 `path`。
