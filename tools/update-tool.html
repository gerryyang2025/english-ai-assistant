        📖 使用方法

        1. 选择数据类型：点击上方标签切换"单词数据"、"阅读数据"或"听书数据"

        2. 上传文件：点击上方区域或拖拽对应的 .md 文件

        3. 预览数据：查看解析后的数据结构是否正确

        4. 生成数据：点击按钮生成 JSON 数据

        5. 更新文件：将生成的文件复制到项目对应位置

        📂 需要更新的文件

        单词数据：

        data/words.json js/app.js (EMBEDDED_WORD_DATA 常量)

        阅读数据：

        data/readings.json js/app.js (EMBEDDED_READING_DATA 常量)

        听书数据：

        data/listen.json js/app.js (EMBEDDED_SPEECH_DATA 常量)
        💡 提示

        • 如果使用本地服务器（http://），只需更新对应的 data/*.json 文件

        • 如果直接打开 HTML 文件（file://），需要同时更新 js/app.js 中的内嵌数据
